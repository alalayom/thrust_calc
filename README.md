# thrust_calc

HX711-based load cell thrust stand data acquisition and analysis project.

## Contents

- Reading load cell data with Arduino
- Data collection from serial port with Python
- Saving to CSV
- Generating thrust-time graphs
- Calculating peak thrust and total impulse

## Folder Structure

- `arduino/` : Arduino sketch files
- `python/` : Python data collection and analysis scripts
- `data/` : CSV outputs
- `docs/` : Notes and test documents
